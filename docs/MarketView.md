### 闪购超市接口

#### 请求地址：/api/goods/market/

#### 请求方式：GET

#### 请求参数：

        goods_list：商品信息  typeid商品分类id   food_type 商品类型
        
        order_rule_list：排序信息
        
        foodtype_childname_list：全部分类中的子分类信息
        
#### 响应

#### 1.响应成功

    code：200，
    msg：'请求成功'，
    data:{}
    

    
#### 响应参数

    code  状态码  int
    
    msg  响应信息 string
